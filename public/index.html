REST API Introduce
feel free to use and contribute!

GET /tagsList

get all tags

Parameter Type Usage Optional
none none none none

Response format: JSON

Response Example:

{ "status": 200, "response": [{ "name": "Department", "value": "1", "children": [ { "name" : "Department - Category", "value" : "1.1", "children" : [] }, { "name" : "Department - Category", "value" : "1.2", "children" : [] }] }] }

POST /newsList

get featured/subscribed news list

Parameter Type Usage Optional
tags Array tags array containing user's subscription, if only contains "0", return featured news list YES
start_index String or Integer start from which news NO
count String or Integer return maximum numbers of news NO

Response format: JSON

Response Example:

{ "status": 200, "response": [ { "title": "News Title", "link": "News FilePath", "tags": ["1.1", "1.2"], "date": "2014-10-20 10:00:00" }, { "title": "News Title", "link": "News FilePath", "tags": ["1.2"], "date": "2014-10-10 10:00:00" } ] }

GET /*/*.json

get specific news

Parameter Type Usage Optional
none none none none

Response format: JSON

Response Example:

{ "status": 200, "response": { "title": "News Title", "link": "News URL", "date": "2014-10-20 10:00:00", "imgs": [ "Image URL", "Image URL" ], "tags": [ "1.1", "1.2" ] } }

POST /push

configure push notification

Parameter Type Usage Optional
token String device UUID YES
action String "register" represent register or update a user's subscription for push notifications, "remove" represent remove this user from push notifications YES
tags Array tags array containing user's subscription NO

Response format: JSON

Response Example:

{ "status": 200 }
